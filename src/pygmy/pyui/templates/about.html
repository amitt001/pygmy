Pygmy: An open-source URL shortener.

Written in Python using Flask and Django framework.
It uses SqlAlchmey as the ORM so any relational database that SqlAlchmey uses can be used. Tested with Postgresql and Sqlite.

The project as 3 parts:

  1. The core program for URL shortening
  2. REST API on top. Uses Flask framework.
  3. The UI layer for rendering the UI. It uses Django framework.

The project is created keeping in mind that each of the part can be used separately without any dependecy on another part.
